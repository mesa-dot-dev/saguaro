# Changelog

All notable changes to this project will be documented in this file.

## [0.3.8] - 2026-03-06

### Added

- Initial open-source release
- AI code review engine with rules enforcement
- Support for Anthropic (Claude), OpenAI (Codex), and Google (Gemini) models
- Claude Code integration (MCP server, stop hook, slash commands)
- Import graph indexing for cross-file context
- Rule generation from codebase analysis
- Review daemon for background operation
- Interactive TUI
- Homebrew, npm, and GitHub Releases distribution
